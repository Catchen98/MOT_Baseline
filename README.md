
This is a small improvement on MOTDT for multi-target tracking, where we use short-time flows refine
the lastest detections.

About the MOTDT, refer to [MOTDT](https://github.com/longcw/MOTDT)

#### Dependencies

- python3.6
- pytorch 0.3.1
- motmetric
- json
- cython
- and so on
